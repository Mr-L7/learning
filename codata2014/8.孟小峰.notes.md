## Big Data vs. Database

* Big Data -> Extremely Large Database (XLDB) -> 数据思维

### 数据产生的演化

运营式系统  - 数据库 （Database）
* 被动产生、规范、有序的

互联网系统  -  数据空间（Dataspace）
* 主动、复杂、无序
* 满足分布式处理和数据集成

感知式系统   -  大数据（Big Data）
* 自动产生
* 数据呈现3V、4V
* 满足流式处理和实时分析
